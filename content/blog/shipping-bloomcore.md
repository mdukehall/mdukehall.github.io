# Shipping BloomCore

Built a card-and-grid strategy game with Expo and React Native. The family wanted to play without installing anything, so the web export went to GitHub Pages.

## Lessons learned

- Static hosting works fine for a game if you plan for the subpath (`/BloomCore-Mobile/`)
- GitHub Pro unlocks Pages on private repos — the **site URL is still public**, but the source stays yours
- Visual feedback matters: siphon clears, trump indicators, live score — players need to *feel* what's happening
